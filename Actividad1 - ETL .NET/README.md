# Actividad 1 - Proceso de Extracción ETL en .NET

Worker Service desarrollado en .NET 8 que extrae datos de forma paralela 
desde tres fuentes distintas: un archivo CSV, una base de datos relacional 
y una API REST. Cada fuente tiene su propio extractor y todos corren 
al mismo tiempo usando Task.WhenAll().
